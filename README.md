# NFX3
this is a bunch of effects with rather messy code compiled into one notsobot tag, just like my first NTFX+ and NitroFXV2. V2 kinda flopped tho
# PRECAUTIONS
> [!WARNING]
> vocoder effects are NOT PRESENT in the bot!
>
> if you see a "geq" code with waves, ALWAYS use ".resize 640" before executing NFX3!
>
> this project is pointlessly marked as GPL 3.0. pastebin exists anyway and the source code will always be there.

# HOW DO I REQUEST AN EFFECT?
send it on the "Issues" tab if you have GitHub.

# WHAT DOES THE BOT USE?
FFmpeg and that's it. yup, purely FFmpeg. this is so you can have a **CLEAN** IHTX without the audio clipping as much as in iscript.

# WHAT SUBFUNCTIONS ARE THERE?
- raw: to show the code of an effect
- help: shows this readme
- source: sends the link to the source code repository (this one)
